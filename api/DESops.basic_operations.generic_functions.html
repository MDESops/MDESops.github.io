DESops.basic_operations.generic_functions module

Contains helpful functions used in various operations.

DESops.basic_operations.generic_functions.find_E(S, E)

Check set of graphs S to find all events.

DESops.basic_operations.generic_functions.find_Euc(S, Euc)

Check set of graphs S to find uncontrollable events.

DESops.basic_operations.generic_functions.find_Euo(S, Euo)

Check set of graphs S to find unobservable events.

DESops.basic_operations.generic_functions.find_obs_contr(S, Euc={}, Euo={}, E={})

For set of graphs S, find Euc and Euo if not provided. This way, checks for Euc, Euo as empty sets are done here, rather than at the place this function gets called (slight convenience).
