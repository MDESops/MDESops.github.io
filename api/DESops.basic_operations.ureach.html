DESops.basic_operations.ureach module

DESops.basic_operations.ureach.extended_ureach(x_set, state, g, e, Euo)

Find the extended unobservable reach from a state in g.

e_o: single observable event, set of events e Defined as unobservable reach via unobservable events e in E_uo, followed by the single observable event e_o in E_o Starting from state in g result stored in x_set
